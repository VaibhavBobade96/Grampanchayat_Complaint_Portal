# Grampanchayat_Complaint_Portal
A web applications for villagers to submit and track complaints online, while administrators manage and resolve issues efficiently.

# Features 
1. User registration & Login
2. Complaint Submission
3. Complaint Status Tracking
4. Admin Dashboard
5. Complaint Management

# Tech Stack 
1. HTML
2. CSS
3. Java Script
4. Python
5. SQLite/MySQL
